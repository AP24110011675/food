# Food Delivery Hub

A modern, full-stack food delivery platform inspired by Zomato and Swiggy.

## Features
- ✨ Premium UI with Hero Carousel
- 🍔 Diverse Restaurant Listings
- 🛒 Real-time Cart Management
- 🔐 JWT Authentication
- 💳 Multi-method Payment Support (UPI, etc.)
- 📱 Responsive Design

## Tech Stack
- **Frontend**: React, Vite, Lucide Icons, Modern CSS
- **Backend**: Node.js, Express, MongoDB, Cloudinary
- **Other**: JWT, Bcrypt, Dotenv

## Getting Started
1. Clone the repo
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
